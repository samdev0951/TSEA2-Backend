# The Operating System (OS)

## What is an Operating System

The Operating System is a vital part of a Computer System that bridges the gap between the hardware, the software and the human interacting with it. Without it, your computer doesn't know what to do! The components know how to do their job, but they need instructions in order to do the things they're supposed to.

Some common examples of Operating Systems you may have heard of:

<ul>
- Windows
- Mac OS
- Linux
- Android
- IOS
</ul>

## What is the OS responsible for?

**Hardware:** The Operating System manages the physical components in your Computer so that applications run smoothly without being concerned with the low level complexities of the computer. This involves managing Input / Output from devices such as your keyboard and mouse and delivering the graphics to your screen.

**User Interface:** Provides a way for the user to interact with the computer in a way that is understandable and familiar.

**Process and Memory Management:** Ensures that various applications run smoothly and are allocated the right about of resources for their demands, manages the dedicated amount of memory (RAM) that each process has.

**File System:** Organises and stores files on your HDD (or SSD) so that they can be viewed, edited, moved or deleted efficiently.
