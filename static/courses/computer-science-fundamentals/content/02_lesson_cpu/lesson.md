# The CPU (Central Processing Unit)

The CPU is like the brain of the computer. It is responsible for processing instructions, delivering instructions to the different hardware components, and arithmetic processing. This is the core component that makes a computer... compute. Without this, a device is not a computer. The CPU has it's own ultra-fast storage, known as registers. The storage capacity of registers is very low, they are used for keeping track of very small pieces of information to access on demand such as instruction operands, memory addresses and control flow information. Think of registers like your working memory, the ultra quick access memory you use to process information!
