# The RAM (Random Access Memory)

The RAM (or memory) is like your short term memory. It is responsible for storing temporary information that the CPU will need in the short term. It is a high speed form of storage, however it not as efficient as the CPU registers which are baked into the CPU. The RAM is only populated when the computer is powered. As soon as the power goes, so does the memory. Everything is forgotten unless it is stored elsewhere (next section). These come in a stick form that can be plugged into the RAM slots on your motherboard.
