# What is a Computer?

You may think of a Computer as one of those big black boxes with loud fans that connects to a screen. You are not wrong, this is the typical type of computer that you see in offices and at home, which you are likely using right now to read this page - also known as a Tower or PC.

However, there are actually many more types of computers than you probably realise. What if I told you that phones, microwaves, cars, traffic lights, bank cards, door bells are all computers? Any electronic device with a processor is officially computer! All of the devices in that list, including your computer will have a few or many of the Computer Components you are going to learn about in this lesson.

## What is a Computer Component

A Computer Component is a physical device that makes up a Computer. Each component has its own specific and important purpose, connected with the other components by an ecosystem of wiring (we'll talk about this later). The components of the Computer are often compared with the human body, so we will use this analogy to describe each of the components. So, let's think of each component like an organ in the human body. Note that this module does not cover all of the possible Computer Components, but only the most common and important ones.
