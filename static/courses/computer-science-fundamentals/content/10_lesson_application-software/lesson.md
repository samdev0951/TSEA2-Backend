# Application Software

## What is Application Software

An Application Software (also just known as Software) is a program designed to solve a specific problem or perform specific tasks. It runs on top of System Software (built into the Operating System) which it relies on to interact with the Hardware.

Some examples of Application Software include:

<ul>
- Web Browsers (Google Chrome, Firefox)
- Document Processors (Word, Excel, Powerpoint)
- Games (Minecraft, Roblox, Fortnite)
- IDEs (Visual Studio Code, Intellij)
</ul>

## How does an Application Run

The typical process for running an application is:

<ol>
- The user opens the application with the User Interface
- The Operating System reads the executable file (in machine code)
- The program is loaded into memory (RAM)
- The CPU begins executing the application instructions (it understands the machine code)
</ol>

## How is Application Software Made

Application Software is often created by a team of Software Engineers. It would be incredibly difficult and time consuming for Software Engineers to write the machine code for software, as it is not something that is human readable or easy to keep track of.

Instead, Software Engineers use Programming Languages as a high level form of instructions that are human readable. Instead of writing machine code, programming languages allow the engineers to write instructions using a generic control flow that is universal across most programmiing languages. Think "If X do X", "Do X until X is True". We cover programming languages in our other courses!

Popular Programming Languages:

<ul>
- Python
- C / C++
- HTML / CSS / JavaScript
- PHP
- Java
</ul>
