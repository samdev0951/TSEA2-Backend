### Variables

Simply put, variables store values.

In Python, there aren’t keywords for defining a variable; instead, you type the name of the variable, then the assignment operator, and finally the value that you want to store in the variable. The format is as follows: [VariableName = Value]. An example utilising the format would be: numberOfCats = 5.

### Data Types

You can store any type of data in a variable. The types of variables don’t need to be explicitly defined, as values are inferred at runtime.

Casting - With casting, you can specify a variable’s type. You would use casting when, for example, you want to store a number as a string, so: numberOfCats = str(5). The command “str” stands for string, and anything inside the brackets is turned into a string.

Type - You can get the type of a variable via the command “type(VARIABLE)”. It gets the type of the specified variable within the brackets. Useful for when dealing with errors, often the error is caused by inconsistent types in a calculation.

### Operators

Operators are used to perform operations on variables and their values. They are commonly used when you want a result from the values of several variables combined.

Arithmetic Operators - These types of operators are performed on numerical values. Examples of arithmetic operators include Addition (+), Subtraction (-), Multiplication (*) and Division (/).

You need to make sure that all values are the same type, because performing operations of different types can result in a type error, which prevents the operation from completing.

Assignment Operators - As the name suggests, Assignment Operators are used to assign values to variables. The most basic assignment operator is “=”. You can also add to an existing variable by following this format: [variable] [operator]= [value]. For example, cats += 5 increases the value of the variable ‘cats’ by 5. Here, + is combined with =, which shortens ‘cats = cats + 5’.
