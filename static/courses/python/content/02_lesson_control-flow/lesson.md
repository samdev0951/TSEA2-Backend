### Control Flow

Control flow refers to the sequence in which statements are executed in a program. The default behaviour of this is that lines of code are executed in order, from the first to the last line of code. In Python, there are features to modify the default control flow behaviour, including if, elif, and else.

Control flow: if - If statements are used to evaluate a condition to either true or false. If the condition is true, the code inside the if block is executed; otherwise, the code block is skipped. 

Control flow: elif-else - This is helpful when you want to check multiple expressions for conditions that evaluate to true. For example, you’re building an IQ test, where you want to categorise different results, you can use elif to handle different score ranges and assign categories accordingly. The ‘else’ keyword is used to handle unexpected values or default cases.
