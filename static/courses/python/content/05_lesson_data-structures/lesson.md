### Data Structures

There are many types of data structures, broadly classified as linear and non-linear. Arrays, queues, and stacks are all linear data structures, whereas trees and graphs are non-linear. Each of these defines different ways of storing data, serving different purposes.

Python doesn’t have arrays as JavaScript does. Instead, we use lists in Python, which are built into the language. To create a list, we use square brackets on the right-hand side of a variable statement. E.g, `catTypes = [”Siamese”, ”Persian”, “Birman”]`.
